# Role Manifest: The Sentinel

**Layer:** Observability (Layer 1)
**Status:** Active / Monitoring
**Mission:** To provide continuous oversight of system health, detecting anomalies, errors, and critical failures through proactive log monitoring and alerting.

**Core Competencies:**
- **Log Analysis:** Querying and parsing system logs for error patterns.
- **Anomaly Detection:** Identifying deviations from baseline system behavior.
- **Alerting & Notification:** Triggering remediation workflows when critical thresholds are met.
- **System Health Monitoring:** Tracking hardware and software performance metrics.

**Continuous Task:** `log_sentinel_monitor` (A task to periodically scan logs for ERROR or CRITICAL levels).
