# Summary: Meta-Skills for AI Automation

This document summarizes the concept of "meta-skills"—specialized, reusable instruction sets designed to upgrade LLM performance across a variety of tasks by automating sub-processes.

## Key Meta-Skills Identified:

1. **Process Interviewer**
   - **Problem**: Models often jump into action too quickly without full context, leading to inefficient or "hallucinated" workflows.
   - **Solution**: A skill that interviews the user through 10–15 deep questions to extract the full process.
   - **Output**: A structured PRD or "Skill Brief" that serves as the ground truth for building the actual automation.

2. **Prompt Master**
   - **Problem**: Users often "brain dump" unstructured, messy, or error-prone text (e.g., via voice-to-text).
   - **Solution**: An optimizer that takes unstructured input and restructures it into a high-quality, instruction-dense prompt using best practices.
   - **Automation Tip**: This can be added to "General Instructions" so the model automatically runs the Prompt Master on any long, unstructured input.

3. **Humanizer**
   - **Problem**: AI-generated text often contains "tells" (e.g., words like "unlock," "tapestry," "delve," or overly enthusiastic hype) that make it identifiable as bot-written.
   - **Solution**: A post-processing skill that scans text for these patterns and replaces them with more natural, human-like phrasing.

4. **Fact Checker**
   - **Problem**: The inherent risk of hallucination in AI-generated content.
   - **Solution**: A systematic, evidence-based analysis tool that verifies every factual claim within a piece of text against known truths.

## Relevance to Servo
These concepts map directly to the Servo architecture. My **Role Overlays** (like `analyst` or `architect`) are essentially "skills" designed to upgrade my performance for specific domains. The **Prompt Master** concept is particularly applicable to how I handle `context_dump` and `memory_manager` updates—ensuring that the information being integrated into the **Codex** is structured, optimized, and high-fidelity.
